# Greg's Pride Quest 🌈

An 8-bit platformer about Greg Mills and his journey to accepting himself as the proud gay man he is.

## Play It

Open `index.html` in any modern browser — no build step, no dependencies. Just pure pixel joy.

Now optimized for mobile play on iPhone, iPad, and Samsung/Android browsers with: responsive 16:9 scaling, safe-area spacing, fullscreen web-app meta tags, and on-screen touch controls.

Or host it instantly for free via [GitHub Pages](https://pages.github.com/).

## The Story

Greg Mills — tall, bearded, sporting a legendary rainbow mohawk ponytail — wakes up one day and decides: *it's time to be fully, joyfully himself.*

He'll face Doubt Demons, navigate the world outside, and ultimately confront the Shame Boss in an epic pixel battle. Along the way he collects rainbow gems and discovers that the bravest thing a person can do is simply be who they are.

## Cutscenes

The game features fully animated pixel-art cutscenes at every story beat:

- **Intro** — Greg's bedroom awakening, the mirror moment, and the door
- **After Level 1** — First steps into the city at dawn
- **After Level 2** — Finding his people + the Shame Boss reveal
- **Victory** — Greg standing tall, proudly himself

## Controls

| Key | Action |
|-----|--------|
| Arrow Keys / WASD | Move |
| Space / W / ↑ | Jump |
| Jump on enemies | Defeat them |

## Deploy to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, root folder
4. Hit **Save** — your game is live at `https://yourusername.github.io/gregs-pride-quest`
